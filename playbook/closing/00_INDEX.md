# 🎯 RB Perform — Playbook Closing

Système opérationnel pour préparer chaque call de vente et progresser systématiquement.

## Structure

| Fichier | Quand l'utiliser |
|---|---|
| `01_psychologie_vente.md` | Avant chaque call — relecture rapide des fondamentaux + 14 armes |
| `02_gestion_objections.md` | Avant chaque call — révision framework 5 étapes + prévention |
| `03_methodologie_closing.md` | Référence opérationnelle (paiement in-call, anti no-show, downsell) |
| `04_offre_rb_perform_pro.md` | Référence offre + grille tarif lockée |
| `05_template_call.md` | Template à remplir AVANT chaque appel prospect |
| `06_pre_call_email.md` | Mail pré-call Hormozi-style (3 questions + témoignages) — réduit no-show, pré-engage |
| `07_emails_follow_up.md` | 5 mails follow-up post-call : J+1 / nurture / last call / "trop cher" / lead silencieux |
| `calls/YYYY-MM-DD_prenom_nom.md` | Brief spécifique pour 1 prospect (cf template) + post-call + onboarding plan si closed |
| `calls/_audit_crm_*.md` | Audits CRM périodiques avec top 10 leads scorés |

## Rituel pré-call (15 min)

1. Lire `04_offre_rb_perform_pro.md` (par cœur)
2. Lire le brief du prospect (`calls/...`)
3. Relire 5 armes critiques de `01_psychologie_vente.md` (silence, OK, lent, expertise, détachement)
4. Relire le framework 5 étapes de `02_gestion_objections.md`
5. 5 min respiration profonde, casque audio, environnement clean

## Rituel post-call (10 min)

1. Noter dans le brief du prospect : ce qui a marché / ce qui a foiré
2. Réécouter l'enregistrement (si dispo) sous 24h max
3. Identifier 1 erreur principale à corriger pour le call suivant
4. Mettre à jour le memory CRM (stage : hot / cold / closed_won)

## Objectif

T'es là pour devenir **le boss de la vente**. Pas en lisant, en répétant. Chaque call est un rep.

Mesure : taux de close sur 5 calls glissants. Si > 40%, t'es au bon prix. Si < 30%, soit le pitch soit le prix sont à retravailler.
