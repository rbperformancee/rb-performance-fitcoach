Données programme (JSON)
Workflow :
  1. Exporte ton programme depuis l'app (bouton "Export JSON" ou copie depuis la console)
  2. Colle le JSON ci-dessus
  3. Clique sur Imprimer / PDF
  4. Dans la dialog d'impression, choisis "Enregistrer au format PDF"
⌨ Raccourci : Cmd+P (Mac) / Ctrl+P (Windows)
